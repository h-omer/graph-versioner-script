== JAVA SCRIPT DATABASE FILL FOR NEO4J GRAPH VERSIONER

*Requirements*:

- Java 1.8


*Installation*:

-----------
./gradlew build
-----------


*Run*:

-----------
java -jar build/libs/neo4j-graph-versioner-data-loader-1.0.0.jar
-----------

WARNING: Change the database connection password if different from _password_
